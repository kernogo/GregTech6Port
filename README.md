# Info

This is an unofficial port of GregTech 6 to modern versions of Minecraft.

This repository is **not** an official version of GregTech 6. You can find the official version here: https://github.com/GregTech6/gregtech6

This port was made using the official version of GregTech 6 as a reference. Assets of GregTech 6 have been reused with some modifications.

Big thanks to [GregoriusT](https://github.com/GregoriusT) and contributors involved for the official version of GregTech 6.
